# 图片裁剪组件说明

## Props

| 参数名      | 类型                                     | 默认值        | 说明                                  |
| ----------- | ---------------------------------------- | ------------- | ------------------------------------- |
| src         | `string`                                 | `''`          | 初始图片地址，支持网络图片和 Base64。 |
| mode        | `'rectangle' \| 'circle'`                | `'rectangle'` | 裁剪模式。                            |
| height      | `string \| number`                       | `'300px'`     | 裁剪区域高度。                        |
| crossorigin | `'' \| 'anonymous' \| 'use-credentials'` | `'anonymous'` | 跨域策略配置。                        |

## Events

| 参数名 | 类型 | 默认值 | 说明                             |
| ------ | ---- | ------ | -------------------------------- |
| 无     | `-`  | `-`    | 当前组件未额外对外派发业务事件。 |

## Methods

| 参数名           | 类型                       | 默认值 | 说明                                                        |
| ---------------- | -------------------------- | ------ | ----------------------------------------------------------- |
| confirmCropImage | `() => Promise<ImageInfo>` | `-`    | 通过 `ref` 调用，返回裁剪结果（base64、blob、宽高、大小）。 |
| resetCrop        | `() => void`               | `-`    | 通过 `ref` 调用，重置裁剪框和图片变换状态。                 |
